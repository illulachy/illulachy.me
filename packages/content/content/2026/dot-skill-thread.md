Your best engineer just quit.

Their code is version-controlled. Their knowledge isn't.

dot-skill distills any person into an AI Skill before they're gone.

---

## It's not a chatbot persona

Most "talk to X" tools just prompt an LLM to "sound like" someone.

dot-skill builds a 6-layer personality model:

- Hard rules (non-negotiables)
- Core identity
- Expression patterns
- Decision frameworks
- Interpersonal dynamics
- Self-correction instincts

---

## Three skill families

**Colleague** — Work-focused. Captures their technical standards, workflows, output preferences, and how they communicate in a professional context.

**Relationship** — For partners, family, old friends. Emotional triggers, conflict patterns, repair moves.

**Celebrity** — Public figures, creators, thinkers, fictional characters. Full six-dimension research toolchain.

---

## Data collection is automated

| Source | Method |
|---|---|
| Slack | API pull |
| Feishu | API pull |
| DingTalk | Browser scrape |
| WeChat | SQLite export |
| Email | Parsed |
| PDFs and screenshots | Manual upload |

You feed it what you have. It extracts the patterns.

---

## How it runs

Generated skills run inside Claude Code as slash commands. Also works in Hermes, OpenClaw, and Codex.

Community gallery is live — 100+ skills already published.

---

## How it started

This started as "colleague.skill" — just for coworkers. Hit enough stars and enough edge cases that they rebuilt it as a universal engine. The rename to dot-skill happened two weeks ago.

---

## Real uses people are reporting

- Capturing a mentor's framework before they retire
- Preserving how it felt to talk to a grandparent
- Interacting with a favorite author's decision logic
- Onboarding new hires with a "shadow" of the engineer who built the system
